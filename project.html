About Project

Project name: Fit For Fish
Objective:
  1. ทราบถึงอุณหภูมิของนํ้า ณ ช่วงเวลานั้นๆ
  2. ลดความเสี่ยงของการสูญเสียของปลา
  3. ป้องกันอันตรายที่อาจเกิดขึ้นจากอุณหภูมิที่ไม่เหมาะสมแก่ปลา
Advantage:
  1. ทราบถึงบรรยากาศ ความเหมาะสมของอุณหภูมิ ความเป็นอยู่ของปลา
  2. ช่วยลดความเสี่ยงของการสูญเสียของปลา
  3. ช่วยป้องกันอันตรายที่อาจเกิดขึ้นจากอุณหภูมิที่ไม่เหมาะสมแก่ปลาได้
  4. ใช้งานง่าย เข้าถึงได้ตลอดเวลา

Detail: โครงงานนี้จัดทําขึ้นเพื่อบอกสถานะของอุณหภูมิในนํ้า ณ ช่วงเวลานั้นๆว่ามีอุณหภูมิเท่าไร และสภาพอุณหภูมิเหมาะสมต่อการดํารงชีพของปลาหรือไม่ โดยอ้างอิงค่าจากเซ็นเซอร์วัดอุณหภูมิของนํ้าที่เชื่อมต่อกับ Arduino โดยควบคุมจากภาษา c จากนั้นจะส่งค่าไปเก็บไว้เป็นสถิติที่เว็บ ThingSpeak แล้วจากนั้นจะดึงข้อมูลล่าสุดจาก ThingSpeak ไปแสดงผลบนเว็บไซต์ ดังนั้นโครงงานชิ้นนี้จึงจัดทำขึ้นมาเพื่อคอยช่วยเหลือ คอยรายงานสถานะอุณหภูมิของน้ำให้ผู้ใช้รู้ว่าขณะนี้ อุณหภูมิดังกล่าวเหมาะแก่การดำรงชีพ เหมาะกับสภาพความเป็นอยู่ของปลาหรือไม่ อีกทั้งยังช่วยป้องกันปัญหาจากอุณหภูมิที่ไม่เหมาะสม ซึ่งอาจเป็นเหตุแก่การสูญเสียหรือการเจ็บป่วยของปลาได้อีกด้วย
